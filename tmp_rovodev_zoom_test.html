How Zoom Works

The zoom feature allows you to:

  • Zoom In: Increase document size by 10% increments
  • Zoom Out: Decrease document size by 10% increments
  • Reset: Return to 100% (original size)
  • Fit to Screen: Automatically calculate the best zoom level to view the entire page without scrolling
